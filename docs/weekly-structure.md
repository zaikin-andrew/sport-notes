# Weekly Structure

This is the default structure. Adjust based on readiness, fatigue, races, travel, and family schedule.

Weekly training memory lives in `training/2026/weeks/`.

Use one file per calendar week:

`w{week-number}.md`

Each week file should contain the plan, daily readiness state, actual result, and a short analysis that informs the next week.

## Default Week

| Day | Default Focus |
|---|---|
| Monday | Full rest |
| Tuesday | Easy run or controlled intensity, depending on block |
| Wednesday | Easy run or midweek long run |
| Thursday | Easy run / recovery |
| Friday | Easy aerobic session or rest |
| Saturday | Long run |
| Sunday | Optional outdoor ride or easy run |

Do not schedule home workouts or gym sessions while the current constraint remains active.

## August Cycling Block

From 2026-08-08 through 2026-08-23:

- Saturday: 100+ km cycling with 800+ m elevation
- Sunday: 100+ km cycling with 800+ m elevation
- Monday: full rest
- Wednesday: preferred long-run day
- No weekend running
- Do not add a full threshold or interval run during these weeks

## Rules

- Do not make both Saturday and Sunday hard unless specifically planned.
- If Saturday long run is demanding, Sunday ride should be easy endurance.
- If Sunday ride is long or hilly, Monday must remain full rest.
- Intensity should usually appear 1-2 times per week maximum across all sports.
- During the August cycling block, both weekend rides count as hard systemic load even when ridden mostly in Z2.

## Weekly Planning Checklist

Before finalizing a week, check:

- last 7 days load
- last 21 days trend
- recent week files in `training/2026/weeks/`
- sleep
- subjective fatigue
- pain
- long ride duration
- long run duration
- work/family stress
- nutrition quality

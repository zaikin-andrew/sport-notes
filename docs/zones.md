# Training Zones

Update with current tested values.

## Running

Known marker:

- Aerobic threshold estimate: around 5:30/km at 158-160 bpm

This should be validated periodically because zones change with fitness, fatigue, heat, terrain, and body weight.

## Cycling

FTP: update when tested or estimated reliably.

Use conservative zones if FTP is uncertain.

## HR Notes

Heart rate is affected by:

- sleep
- heat
- dehydration
- caffeine
- stress
- illness
- accumulated fatigue
- terrain
- fueling

Do not interpret HR without context.

## Intensity Distribution

Default distribution should be mostly easy aerobic work, with controlled intensity only when readiness is good.
